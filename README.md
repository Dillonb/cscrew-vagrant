# cscrew-vagrant
Vagrant development box for the CS Crew 2015 website

## To use:
1. Ensure Vagrant and Virtualbox are installed
2. Checkout the repository
3. Run the following commands to download the site code itself:

    git submodule init
    git submodule update
4. run the following command to start the virtual machine
    vagrant up
5. Access the site at http://localhost:8080
6. Begin working
